开发一款仿 Apple Music 风格的 Android 音乐播放器，并结合你维护的 `nimabi520/api-enhanced` (网易云音乐 API) 作为后端，是一个非常棒的项目！

更新：可以通过这个库实现liquid glass效果  https://github.com/Kyant0/AndroidLiquidGlass?tab=readme-ov-file
并且歌词界面可以参考：https://github.com/Yos-X/FlamingoSank/ 还有 https://github.com/amll-dev/applemusic-like-lyrics

为了实现类似 Apple Music 那样流畅、现代化的 UI 和稳定的后台播放体验，强烈建议使用 **现代 Android 开发技术栈**：
*   **开发语言**：Kotlin
*   **UI 框架**：Jetpack Compose (非常适合实现 Apple Music 复杂的交互、高斯模糊和流畅动画)
*   **架构模式**：MVVM (Model-View-ViewModel) + 响应式架构 (MVI/Clean Architecture)
*   **网络请求**：Retrofit + OkHttp + Kotlin Coroutines & Flow
*   **媒体播放**：AndroidX Media3 (ExoPlayer) - 现代 Android 音频播放的标准库
*   **依赖注入**：Hilt

以下是推荐的 Android 项目目录结构，采用了基于功能（Feature-based）和分层（Layered）的混合架构设计：

### 开发建议与重点：

1. **对接 `api-enhanced` 时的注意事项**：
    * 网易云 API 需要处理登录态，建议使用 `Cookie` 或者 `realIP` 处理。在 `data/api/AuthInterceptor.kt` 中统一拦截并加上需要的凭证。
    * 音源 URL 获取：播放时需要通过 `/song/url/v1` 接口实时获取真实的 mp3/flac 链接喂给 Media3 Player。

2. **实现 Apple Music 的 UI 灵魂**：
    * **大标题排版 (Large Typography)**：在页面的顶部使用超大的粗体标题，向下滚动时可以平滑过渡到顶部的 TopBar。
    * **高斯模糊 (Glassmorphism)**：Apple Music 播放界面的灵魂。在 Jetpack Compose 中，可以使用 `RenderEffect.createBlurEffect` (Android 12+) 或者借助第三方库来实现背景随着专辑封面颜色动态变化的实时高斯模糊效果。
    * **动态圆角和动画**：展开底部迷你播放器 (Mini Player) 到全屏播放页面的手势驱动动画，可以使用 Compose 的 `ModalBottomSheet` 或自定义交互实现。

3. **后台播放体验**：
    * 千万不要用旧版的 `MediaPlayer`。一定要使用 **AndroidX Media3** 的 `MediaSessionService` 或 `MediaLibraryService`。它能完美兼容 Android 的后台限制、蓝牙耳机控制、锁屏播放控制以及车载 Android Auto。

### 目前的开发计划
第 1 步“搜索可用”（最优先）
目标：输入关键词，能拿到歌曲列表并显示基本信息（歌名/歌手/封面/id）。
主要文件：app/src/main/java/com/example/seteasecloudmusic/data/api/NeteaseMusicService.kt、app/src/main/java/com/example/seteasecloudmusic/data/model/SongDao.kt、新增仓库实现（可放 data/repository/）。
完成标准（DoD）：固定关键词（比如“周杰伦”）请求稳定返回，UI 能显示列表，不崩溃。
自测方式：连续搜 3 次同关键词；断网时有错误提示；空结果时有空态提示。

做完第 1 步后，马上进入第 2 步“点击可播放”
目标：从列表点一首歌，能获取播放 URL 并开始播放（先不追求完美 UI）。
主要产物：domain/model/Track、domain/usecase/*、player/playback/MusicPlayerController。
DoD：可播放/暂停/继续，异常 URL 不崩溃。
第 3 步才是后台播放（Media3）
目标：切后台不断播，通知栏/锁屏控制可用。
主要产物：player/playback/MusicService + app/src/main/AndroidManifest.xml 服务声明。
DoD：熄屏、切应用、耳机按键都能控。
